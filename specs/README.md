# Specifications - dbt for Streaming

This folder contains all the functional specifications, design decisions, and test cases for the "dbt for Streaming" project.

## Structure

- `VISION.md` - Product vision, goals, and non-goals
- `CONCEPTS.md` - Core concepts and terminology
- `DSL.md` - DSL specification (YAML schema, syntax)
- `COMPILATION.md` - How the compiler works (Proxy, Flink, Connect targets)
- `CLI.md` - CLI commands and user experience
- `SECURITY.md` - Security model and policies
- `TESTS.md` - Streaming tests specification
- `QUESTIONS.md` - Open questions and decisions to make

## Status

Currently in **Discovery Phase** - gathering requirements and validating concepts.
