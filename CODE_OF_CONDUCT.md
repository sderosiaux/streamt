# Code of Conduct

## Our goal

This project exists to build useful software in the open. We want a space where people can work together, share ideas, and disagree without causing harm. We value clear thinking, direct feedback, and respect for people.

## Expected behavior

Everyone taking part in this project agrees to:

- Treat others with respect, even during disagreement.
- Focus on ideas, code, and facts rather than personal traits.
- Communicate clearly and honestly.
- Assume good intent first, then ask for clarification.
- Accept feedback without personal attacks.
- Help keep discussions productive and on topic.

## Unacceptable behavior

The following behavior is not accepted in this project:

- Harassment, intimidation, or personal attacks.
- Discrimination based on personal traits.
- Sexual language, imagery, or unwanted attention.
- Trolling, insults, or repeated disruptive behavior.
- Publishing private information without permission.
- Threats of violence or encouragement of harm.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues and pull requests
- Discussions and comments
- Community chats or forums
- Project events or meetups

## Enforcement

Project maintainers enforce this Code of Conduct. They may take actions such as:

- Asking for a change in behavior
- Removing comments, issues, or pull requests
- Temporarily limiting participation
- Permanently banning contributors in severe cases

Maintainers act in good faith and aim for fair outcomes.

## Reporting issues

If you experience or witness behavior that breaks this Code of Conduct, raise a GitHub issue.

When possible, include:
- A clear description of what happened
- Links to relevant comments or pull requests
- Any context that helps maintainers understand the situation

Maintainers will review reports and respond as quickly as possible.
